# gestion_donnees_boutique
Traitement et nettoyage des données de stock d'une boutique

Objectifs :

    Effectuer des analyses univariées et multivariées des données pré-traitées
    Pré-traiter des données pour explorer et comprendre leurs caractéristiques
    Traiter, nettoyer des données et définir la gestion des incohérences en conformité du RGPD

Compétences :
    
    Travailler avec un notebook Python
    Effectuer des analyses exploratoires de fichiers
    Effectuez des jonctions de fichiers
    Effectuez des analyses univariées
